# scGI
Official implemetntation of scGI

#### Abstract
We present scGI, a machine learning framework for identifying disease-relevant gene-gene interactions from single-cell RNA sequencing (scRNA-seq) data. Built on factorization machines, scGI explicitly models pairwise gene interactions in high-dimensional, sparse single-cell data. To improve robustness and generalization, scGI incorporates an invariant loss that enforces interaction consistency across cell types and an embedding decorrelation regularizer that suppresses noisy correlations. We apply scGI to scRNA-seq datasets from Alzheimer’s disease (AD) and breast cancer (BC), where it outperforms five baseline methods and generalizes effectively to unseen cell types in zero-shot settings. Pathway analyses demonstrate biological relevance: in AD, inferred interactions implicate microglial activation, amyloid processing, calcium signaling, and synaptic dysfunction; in BC, interactions enriched for ribosomal and mitochondrial biogenesis highlight metabolic adaptations in tumor and microenvironmental cells.
